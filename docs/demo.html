Votyra demo

Please ensure hardware acceleration is enabled in browser.

Use mouse to select tool and then left click on the terrain to apply.

Use Shift to increase the tools range

Use Ctrl or Right click to invert the tool

Sadly WebAssembly and Unity WebGL do not support Threads yet, this means whole computation has to be done in this demo on the main thread. On platforms supporting Threads most of the work has been offloaded to other threads, only Unity specific parts remain on main thread (gather initial state and set Mesh object). You can check out the repository itself in the Editor for more or download a Windows client here.
