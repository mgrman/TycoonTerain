Votyra

Votyra is a middleware designed to generate 'Transport Tycoon' style terrain in Unity 3D. Currently the terrain is generated based on plane but will be expanded to voxels. The generation is fast and can be run on separate thread, as not to hinder performance. The whole terrain can be animated and can be modified during in runtime.

Use mouse to select tool and then click on the terrain to apply.

Use CTRL to increase the tools range
